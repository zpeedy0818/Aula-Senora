# Prototipado Visual (Mobile First) - Aula Señora 🎓

Este documento presenta la propuesta visual móvil para la plataforma **Aula Señora**, optimizada para una futura integración nativa en **Android Studio**.

## Paleta de Colores (Sistema Ocean & Aurora)

*   **Primario (Azul Profundo):** `hsl(221, 83%, 30%)`
*   **Secundario (Teal/Cian):** `hsl(180, 70%, 45%)`
*   **Fondo (Blanco Hielo):** `hsl(210, 40%, 98%)`
*   **Tipografía:** *Inter*

## 1. Inicio en Dispositivos Móviles (Home)

La vista móvil prioriza el acceso rápido a cursos y perfil mediante una barra de navegación inferior (Bottom Navigation Bar), facilitando el uso con una sola mano.

## 2. Inicio de Sesión (Login)

Pantalla optimizada con objetivos táctiles grandes (min 48dp) y soporte para teclados virtuales específicos.

![Mockup Mobile Login](file:///c:/Users/Ocada/Proyectos/AulaSenora/Documentación/Mobile_Login_Mockup.png)

## 3. Panel del Estudiante (Dashboard)

Diseño basado en tarjetas interactivas que muestran el progreso de los cursos y las actividades pendientes de manera compacta.

![Mockup Mobile Dashboard](file:///c:/Users/Ocada/Proyectos/AulaSenora/Documentación/Mobile_Dashboard_Mockup.png)
